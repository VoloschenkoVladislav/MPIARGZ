# MPIARGZ

Класс gameBoard:
	gameBoard(int m, int n, int x, int y)
		Инициализатор класса. Создает игровое поле m*n точек.
		Левый верхний угол располагается по коорлинатам (x, y).
	char gameTurn(int x, int y, char side, char player)
		Метод хода. Соединяет точки в стороне side от клетки (x, y).
		Устанавливает маркер игрока player в клетке (иными словами player - игрок, совершающий ход), если она огорожена. 
		Проверяет игровое поле на заполненность.
		side может принимать значения 'u', 'd', 'l', 'r'.
		player может принимать значения 'x', 'o'.
		Выбрасывает ошибки:
			0 - неверный аргумент;
			1 - выход за пределы поля;
			2 - место для отрисовки линни уже занято.
		Возвращает значения:
			'n' - ход прошёл без особых событий;
			'x' - игрок Х заполнил клетку;
			'o' - игрок О заполнил клетку;
			'l' - игрок О победил;
			'r' - игрок Х победил.			
	void draw()
		Отрисовывает игровое поле.
